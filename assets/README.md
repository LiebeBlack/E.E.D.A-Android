# Assets de Isla Digital

Esta carpeta contiene los recursos visuales y de audio de la aplicación.

## Estructura

- `images/` - Imágenes de fondo, personajes, escenarios
- `icons/` - Iconos de insignias y botones
- `audio/` - Efectos de sonido y música
- `animations/` - Animaciones Lottie
- `fonts/` - Fuentes tipográficas
