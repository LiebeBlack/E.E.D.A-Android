# Assets de Animaciones

Esta carpeta contiene las animaciones Lottie utilizadas en la aplicación.

## Estructura sugerida:

- `celebrations/` - Animaciones de confeti, trofeos, éxito
- `characters/` - Animaciones de personajes (ondas, saltos)
- `ui/` - Animaciones de carga, transiciones

## Formato:
- JSON (Lottie format)

## Recursos recomendados:
- LottieFiles.com (gratuitos y de pago)
- Crear animaciones personalizadas con After Effects + Bodymovin

## Animaciones necesarias:
1. `celebration.json` - Confeti y celebración al completar nivel
2. `badge_unlock.json` - Animación al ganar insignia
3. `wave_loading.json` - Loading con tema de olas
4. `palm_sway.json` - Palmera moviéndose suavemente
