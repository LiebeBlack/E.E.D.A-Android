# Assets de Audio

Esta carpeta contiene los archivos de audio de la aplicación.

## Estructura sugerida:

- `music/` - Música de fondo (ambiente caribeño, relajante)
- `sfx/` - Efectos de sonido (botones, éxito, error, celebración)
- `voice/` - Instrucciones de voz (opcional)

## Formatos recomendados:
- MP3 para música (calidad 128-192kbps)
- OGG para efectos de sonido (mejor compresión)
- WAV solo si es necesario para calidad sin pérdida

## Nota cultural:
Incluir elementos sonoros de Nueva Esparta:
- Tambores de tamborileros
- Sonidos de olas del mar
- Aves tropicales (guanaguanare, cotorras)
- Viento suave de la isla
