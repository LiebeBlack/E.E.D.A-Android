# Assets de Imágenes

Esta carpeta contiene las imágenes utilizadas en la aplicación Isla Digital.

## Estructura sugerida:

- `backgrounds/` - Fondos de pantalla (atardeceres, playas, cielo)
- `characters/` - Avatares y personajes
- `icons/` - Iconos de la app y UI elements
- `badges/` - Insignias y medallas del sistema de gamificación

## Formatos recomendados:
- PNG para imágenes con transparencia
- JPG para fotografías
- SVG para iconos vectoriales (opcional con flutter_svg)

## Nota cultural:
Incluir elementos visuales de Nueva Esparta:
- Castillo de Santa Rosa
- Cactus Cardón
- Palmeras
- Atardeceres de Juan Griego
- Playas de El Yaque
