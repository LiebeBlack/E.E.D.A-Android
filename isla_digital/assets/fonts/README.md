# Fonts

Esta carpeta contiene las fuentes tipográficas utilizadas en la aplicación.

## Fuente Principal: Nunito

La aplicación utiliza **Nunito** como fuente principal por ser:
- Redonda y amigable para niños
- Altamente legible en diferentes tamaños
- Disponible en Google Fonts (open source)

## Archivos requeridos:

- `Nunito-Regular.ttf` - Peso normal (400)
- `Nunito-Bold.ttf` - Negrita (700)
- `Nunito-ExtraBold.ttf` - Extra negrita (800) para títulos

## Descarga:
Descargar desde Google Fonts:
https://fonts.google.com/specimen/Nunito

## Licencia:

## Alternativa:
Si no se pueden incluir las fuentes personalizadas, usar `google_fonts` 
package que carga las fuentes automáticamente desde la web.
